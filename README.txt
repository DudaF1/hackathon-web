RH Cosmetics - Admin Web

Como usar:
1. Deixe os arquivos index.html, styles.css e app.js na mesma pasta.
2. Abra index.html no navegador.
3. Para editar dados iniciais, altere as constantes no começo do app.js.

Esta versão usa HTML, CSS e JavaScript puros, sem dependências externas.
Os dados são salvos no localStorage do navegador.
